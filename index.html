A multipurpose Discord bot for the mobile rhythm game HATSUNE MIKU: COLORFUL STAGE!

Sekaimiru is a database bot for the game, able to retrieve all kinds of information you need and also featuring fun commands like games and profile customization!
It is powered by another fan-made project called Sekai Viewer. Please support the Sekai Viewer team on the website!

Sekaimiru can be invited here, and you can also join the support server for questions and anything else!
